# 給与明細かんたん計算

## アプリ名

**給与明細かんたん計算**

---

## 目的

紙の給与明細に手書き・手入力している数字をWeb画面に入力し、**A〜Gの給与計算項目を自動計算**する簡易ツールです。家族や社内の事務担当者が、毎月の確認作業を素早く行えることを目指しています。

> **重要:** 本アプリは**正式な給与計算システムではありません**。社内確認用の簡易計算ツールです。確定した給与・税・社会保険料は、必ず会社の規定・既存の手続きに従って確認してください。

---

## 解決する不（ペイン）

| 不 | 本アプリでの対応 |
|----|------------------|
| 紙での手計算が面倒 | 入力するとA〜Gを即時表示 |
| 計算ミスが起きやすい | 計算式をコードで固定し、毎回同じロジックで算出 |
| 毎月同じ項目の確認が手間 | 画面上で同じ入力欄を繰り返し利用 |
| 合計・控除・差引支給額を毎回足し算したい | A〜Gを自動計算し、記録・CSV出力に対応 |

---

## 使い方

### 1. セットアップ（初回のみ）

```bash
cd salary-calculator
python3 -m venv .venv
source .venv/bin/activate          # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

### 2. 起動

```bash
streamlit run app.py
```

ブラウザで **http://localhost:8501** を開きます。

### 3. 操作の流れ

1. **基本情報**（月分・氏名・作成日）を入力する
2. **支給項目**・**非課税通勤**・**社会保険**・**税・控除**に、紙の明細の金額を入力する（0以上）
3. **自動計算結果**で A〜G を確認する
4. **「この内容を記録する」**を押す（氏名は必須）
5. **記録一覧**で過去の入力を確認し、必要なら **CSVでダウンロード**する
6. 一覧を空にしたいときは **「記録をクリア」**を押す

### 4. 記録について

- 記録はブラウザ内の `st.session_state` に一時保存されます
- **DBには保存しません**（第一弾）
- タブを閉じたり、セッションが切れると記録が消える場合があります

## 自動計算される項目

画面上では `st.metric` で次の7項目を表示します。

| 記号 | 項目名 |
|------|--------|
| A | 給与総額 |
| B | 非課税通勤 |
| C | 課税対象給与 |
| D | 社会保険料計 |
| E | 差引控除後給与額 |
| F | 控除計 |
| G | 差引支給額 |

---

## 計算式

```
A 給与総額         = 基本給 + 時間外手当 + 通勤手当 + その他手当1 + その他手当2 + その他手当3
B 非課税通勤       = 入力値（そのまま）
C 課税対象給与     = A − B
D 社会保険料計     = 健康保険 + 厚生年金 + 雇用保険 + その他社会保険
E 差引控除後給与額 = A − D
F 控除計           = 所得税 + 市町村民税 + その他控除
G 差引支給額       = E − F
```

## 注意書き

```
※このアプリは社内確認用の簡易計算ツールです。
  正式な給与計算・税額・社会保険料は会社の規定に従って確認してください。
```

- 計算結果は**参考値**です。給与支払・申告・社保手続きの公式な根拠には使わないでください。
- 法令・社内規程・算定基礎の変更には追随しません。運用前に担当者が必ず検算してください。
